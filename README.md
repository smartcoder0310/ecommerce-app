### Get Started


#### Running server:
```
npm install
COSMIC_BUCKET=your-bucket-slug READ_KEY=your-read-key WRITE_KEY=your-write-key npm start
```

#### Building Ecommerce App:
```
gulp
```
Then go to [http://localhost:3000](http://localhost:3000)
